# Financial Calculator

A Java console application that performs common financial calculations including mortgage payment estimation and present value analysis.

Built to practice financial formula implementation and structured console-based application design.

## Features

- Mortgage payment calculator
- Present value (ordinary annuity) calculator
- Menu-driven interface
- Formatted currency output
- Input validation to prevent crashes

## Mortgage Calculator

Calculates:
- Monthly loan payment
- Total interest paid over the life of the loan

Inputs:
- Loan amount
- Annual interest rate
- Loan term (years)

Uses the standard amortized loan formula.

## Present Value Calculator

Calculates:
- Present value of an ordinary annuity

Inputs:
- Monthly payout
- Annual interest rate
- Term in years

## Technical Concepts Demonstrated

- Object-oriented structure
- Console-based application design
- Mathematical formulas using Math.pow
- Input handling with Scanner
- Clear separation of calculation logic

## How to Run

Compile and run the Main class from the command line or any Java IDE.
